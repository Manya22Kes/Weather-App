# 🌦️ Weather App

A clean and responsive weather application that provides real-time weather updates using the OpenWeatherMap API.

---

## ✨ Features

* 🔍 Search weather by city
* 📍 Use current location (Geolocation API)
* 🌡️ Toggle between Celsius and Fahrenheit
* 📊 5-day weather forecast
* 🎨 Dynamic background based on weather
* ⚡ Smooth UI with loading states
* 🌐 Offline detection & error handling
* 📱 Fully responsive (mobile + landscape)

---

## 🛠️ Tech Stack

* HTML
* CSS (Glassmorphism + Responsive Design)
* JavaScript (Vanilla JS)
* OpenWeatherMap API

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/weather-app.git
   ```

2. Open `index.html` in your browser
   *(or use Live Server for best results)*

---

## 🔑 API Setup

This app uses OpenWeatherMap API.

1. Get your API key from: https://openweathermap.org/api
2. Replace in `app.js`:

   ```js
   const API_KEY = 'your_api_key_here';
   ```

---

## ⚠️ Notes

* Geolocation works only on **HTTPS or localhost**
* API key is exposed (frontend app limitation)
